Expresiones regulares usadas para validar un formulario

Podemos validar un formulario de una manera mucho más personalizada si ocupamos expresiones regulares puesto que estas nos permiten reconocer cualquier patrón, así al no activarse la expresión regular significaría que el usuario aún no ha ingresado un texto válido.

Expresiones regulares usadas para encontrar coincidencias en textos

Una forma muy útil para el uso de expresiones regulares es el contar las palabras que coinciden con nuestra expresión dentro de un texto cualquiera, a modo de análisis.

¿Que es una regex?

Es una secuencia de caracteres que define un patrón de búsqueda. Este patrón es usado para encontrar coincidencias dentro de cadenas, ayudarte a identificar texto específico o patrones de caracteres, proveyendo una forma poderosa de buscar, reemplazar, y manipular texto.
